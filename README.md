# Console CGPA Calculator

A simple console-based CGPA calculator built with C# and .NET 8.

## Features

* Input multiple courses
* Calculate grades based on scores
* Compute total GPA
* Display formatted results in a table

## Technologies

* C#
* .NET 8
* Console Application

## How to Run

1. Clone the repository
2. Open in Visual Studio
3. Run the project

## Example Output

* Displays course list, grades, and GPA calculation
